ROS2 Robot Application

This repository contains ROS2 packages for robot control and manipulation, including:

  • grasp_training: Packages for training grasping algorithms
  • domain_randomization: Tools for domain randomization in simulation
  • kinova_scene_demo: Demo scenes for Kinova robot arms
  • ros2_kortex: Kinova Kortex ROS2 drivers and utilities

Setup

Clone this repository into your ROS2 workspace:

git clone https://github.com/zxjAiLun/robot-app.git
cd robot-app
colcon build

Usage

Launch the robot demo:

ros2 launch grasp_training grasp_demo.launch.py

Grasp Training

The grasp training package provides tools for training and evaluating robotic grasping algorithms. It includes utilities for data collection, training, and evaluation in both simulation and real-world environments.

Domain Randomization

Domain randomization helps bridge the sim-to-real gap by randomizing visual and physical properties of the simulation environment. This package includes tools for randomizing textures, lighting, dynamics, and other properties.

Kinova Scene Demo

This package provides demonstration scenes for the Kinova robot arms. It includes configurations for common tasks and environments for testing your algorithms.
